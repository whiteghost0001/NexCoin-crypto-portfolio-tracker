## 🔍 Quick Reference

### Commands
```bash
# Setup
./setup.sh

# Development
cd server && npm run dev
cd client && npm run dev

# Seed data
cd server && npm run seed

# Docker
docker-compose up -d
docker-compose logs -f
docker-compose down

# Production
npm run build
pm2 start server.js
```

### URLs
- Frontend: http://localhost:3000
- Backend: http://localhost:5000
- MongoDB: mongodb://localhost:27017
